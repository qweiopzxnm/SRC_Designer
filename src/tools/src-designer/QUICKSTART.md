# 快速开始指南

## 🚀 新增功能概览 (v1.1.0)

### 1. 励磁电感 Lm 参数
现在可以在设计参数中直接设置励磁电感 Lm（默认 400μH），系统会自动计算电感比 k = Lm/Lr。

### 2. 电感比 k 值显示
在实际选定参数部分新增 k 值显示，帮助评估变压器设计是否合理（典型范围 3-6）。

### 3. PLECS 仿真导出
点击"🔌 导出 PLECS 参数"按钮，一键生成 JSON 格式的仿真参数文件。

## 📝 使用流程

### 步骤 1：设置设计参数

在输入面板中设置以下参数：

| 参数 | 说明 | 示例值 |
|------|------|--------|
| Vin_max | 最大输入电压 | 810 V |
| Vo_nom | 额定输出电压 | 680 V |
| Po | 输出功率 | 11000 W |
| fr | 谐振频率 | 130 kHz |
| Np | 原边匝数 | 24 |
| Ns | 副边匝数 | 23 |
| Q | 品质因数 | 0.62 |
| fs_ratio | 最小频率比 | 1.2 |
| C_unit | 单颗电容容量 | 47 nF |
| L_step | 电感分辨率 | 1 μH |
| **Lm** | **励磁电感** | **400 μH** ⭐新增 |

### 步骤 2：执行计算

点击 "🔧 自动优化计算" 按钮，系统会自动：

1. 计算设计参数 (Dsnpara)
2. 计算实际选定参数 (Actpara)，包含 k 值
3. 估算效率
4. 生成设计报告

### 步骤 3：查看结果

结果分为三个部分：

#### 设计参数 (Dsnpara)
- 电压增益 M
- 变压器匝比
- 总增益 Gain
- 等效交流电阻 Rac
- 反射电阻 Racp
- 特征阻抗 Zr
- 谐振电感 Lr
- 谐振电容 Cr
- 最小开关频率 fs_min
- 谐振腔电压 Vintank
- 谐振电流峰值 Irpk

#### 实际选定参数 (Actpara) ⭐已更新
- 原边谐振电容 Cr_p
- 原边电容并联数量
- 副边谐振电容 Cr_s
- 副边电容并联数量
- 原边谐振电感 Lr_p
- 等效电容 Ceq
- 设计电容 Cr
- Ceq 偏离度
- 实际 Q 值
- 实际谐振频率
- **电感比 k (Lm/Lr)** ⭐新增

#### 效率估算
- 总损耗
- 估算效率

### 步骤 4：导出结果

#### 导出设计报告
点击 "📥 导出报告" → 生成 Markdown 格式完整报告

#### 导出 PLECS 参数 ⭐新增
点击 "🔌 导出 PLECS 参数" → 生成 JSON 格式仿真参数文件

## 🔌 PLECS 仿真集成

### 方法一：手动导入（立即可用）

1. **在 LLC 设计工具中**：
   - 完成参数计算
   - 点击 "🔌 导出 PLECS 参数"
   - 保存生成的 JSON 文件（如 `PLECS_Params_2026-03-27.json`）

2. **在 PLECS 中**：
   - 打开你的 LLC 仿真模型
   - 使用 MATLAB Script 读取 JSON 文件：
     ```matlab
     params = jsonread('PLECS_Params_2026-03-27.json');
     
     % 设置模型参数
     Lr = params.Lr;
     Cr = params.Cr;
     Lm = params.Lm;
     Np = params.Np;
     Ns = params.Ns;
     % ... 其他参数
     
     % 运行仿真
     sim('llc_model');
     ```

3. **导出仿真结果**：
   - 在 PLECS 中提取效率、波形等数据
   - 保存为 JSON 或 MATLAB 格式
   - 可手动对比设计值和仿真值

### 方法二：自动集成（需要配置）

详细配置步骤请参阅 `PLECS_INTEGRATION.md`，包括：

- JSON 文件格式定义
- PLECS 模型脚本接口
- 命令行自动仿真
- 结果自动读取

## 💡 使用技巧

### 1. 优化电容配置
如果 "Ceq 偏离度" 显示红色（>10%），尝试调整单颗电容容量：
- 点击提示中的推荐值
- 重新计算直到偏离度<5%（绿色）

### 2. 调整电感比 k
- k 值典型范围：3-6
- k 值越小：增益范围宽，但峰值电流大
- k 值越大：效率高，但增益范围窄
- 通过调整 Lm 来改变 k 值

### 3. 参数本地保存
- 所有参数自动保存到浏览器本地存储
- 下次打开页面自动恢复上次设置
- 点击 "🔄 重置" 恢复默认值

### 4. 快速计算
- 在任意输入框按回车键即可触发计算
- 无需每次都点击按钮

## ❓ 常见问题

### Q: Lm 值应该设为多少？
A: Lm 典型值为 Lr 的 3-6 倍。例如 Lr=40μH，则 Lm 可在 120-240μH 之间选择。具体取决于：
- 输入电压范围（宽范围选小值）
- 效率要求（高效率选大值）
- 变压器尺寸限制

### Q: k 值显示红色怎么办？
A: k 值本身没有好坏，关键看是否满足你的设计需求。如果 k 值超出 3-6 范围，考虑：
- 调整 Lm 值
- 重新选择变压器匝比
- 优化谐振频率

### Q: 导出的 PLECS 参数如何使用？
A: 最简单的方法：
1. 在 PLECS 中添加 MATLAB Script 块
2. 使用 `jsonread()` 读取导出的 JSON 文件
3. 将参数赋值给模型变量
4. 运行仿真

详细方法见 `PLECS_INTEGRATION.md`。

### Q: 为什么删除了器件应力和增益曲线？
A: 根据用户需求简化界面。如需这些功能：
- 器件应力：可参考设计报告中的估算值
- 增益曲线：可通过 PLECS 仿真获得更准确的结果
